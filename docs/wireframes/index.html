Seatly

Asaan sawari — know before you board

GSRTC-first bus awareness demo. Passengers see line-up and occupancy; conductors sell from the seat map. Occupancy stays honest because it comes from tickets, not manual overrides.

Not affiliated with GSRTC. Demo data and simulated UPI only. Design source for developers — naming matches Flutter production widgets.

How this app works

  1. Choose a role below (or tap bottom tabs in the phone).
  2. Passenger: From → To → service list → detail → PNR ST9X2K1 on EXP-1101 → Track bus.
  3. Conductor: Pick trip → seat grid → tap free seat → payment (UPI / Cash) → seat occupied. Standing unlocks when full.
  4. Toggle Dev panel (⚙) for RouteName, widget, file path, and Provider.
