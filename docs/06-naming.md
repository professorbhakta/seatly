# Naming

## Working name: Seatly

**Seatly** is the temporary product name for repo, demo app, and conversation.

Do not block v0.1 on a final brand decision. Ship the workflow and rules first.

## Tagline (Indian voice)

**Asaan sawari — know before you board.**

- *Asaan sawari* — easy journey / easy ride (Hindi–Gujarati friendly direction)
- English clarity for “know before you board”

The tagline carries regional voice even if the app name stays English for now.

## Why not Chalo

“Chalo” is an established city-bus brand. Using it would confuse users and misposition Seatly. **Do not use Chalo** as a name or positioning hook.

## Crowded names (caution)

| Name | Issue |
|------|-------|
| **Seatly** | Restaurants, events, other seating apps |
| **OpenSeat** | Tickets, carpool, venues |

Proceed with Seatly; revisit before public launch if search or trademark is a problem.

## Alternatives on the shelf

Not chosen; kept for later rename discussion:

| Name | Note |
|------|------|
| **Baithly** | Seat/baith wordplay; regional feel |
| **Asanly** | From *asaan* (easy) |
| **ClearBus** | Descriptive; less distinctive |
| **Seatly Bus** / **Seatly ST** | Disambiguate from non-transit “Seatly” products |
